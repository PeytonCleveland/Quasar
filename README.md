## About Quasar

![Quasar](/assets/quasar.png)

Quasar is an open-source platform to orchestrate containerized ML workloads on any cloud provider with minimal fuss. I created Quasar out of a need to simplify the process of running both short-lived jobs, such as model benchmarking, and long-running services, such as inference servers, that require GPU resources. Using Quasar is as simple as pushing your container to a registry and then creating a job or service in the Quasar UI, CLI, or API. 

Features:
* Quasar is designed to be simple and easy to use.
* Quasar is designed to be flexible and customizable.
* Quasar is designed to be scalable and robust.
